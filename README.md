# Personal Portfolio

An interactive and modern personal portfolio website built to showcase my projects, technical skills, and creative web development experiments.

This portfolio focuses on clean UI, smooth animations, and engaging user interactions, providing visitors with a fun way to explore my work and development journey.

---

## Live Website

Visit the portfolio

https://my-fun-portfolio-ten.vercel.app/

---

## Overview

This project is a personal portfolio website designed to highlight:

- My development projects  
- Technical skills and tools  
- Interactive web design experiments  
- Contact and networking opportunities  

The portfolio emphasizes performance, responsive design, and modern frontend practices.

---

## Features

- Interactive and fun UI experience  
- Responsive design for desktop, tablet, and mobile  
- Smooth animations and transitions  
- Projects showcase section  
- Skills and technologies display  
- Contact section for collaboration  
- Fast build and development workflow using Vite  
- Component-based architecture for scalability

## Installation and Setup

Clone the repository

```bash
git clone https://github.com/your-username/portfolio.git
```

Move into the project directory

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

The application will start locally.

---

## Build for Production

Create the production build

```bash
npm run build
```

Preview the build

```bash
npm run preview
```

## Customization

You can personalize the portfolio by:

- Updating personal information  
- Adding your own projects

- Editing components inside the `src/components` folder
